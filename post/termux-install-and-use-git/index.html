Termux安装使用git

By 是琪琪呢 | 2023, 08-30 | 1 minutes
termux git教程 折腾

开始安装git

1

pkg install git

配置一下用户信息

1
2

git config --global user.name "你的git用户名"
git config --global user.email "注册git使用的邮箱"

安装ssh

1

pkg install openssh

报错可以尝试安装一下openssl

1

pkg install openssl

生成key

1

ssh-keygen -t rsa -C "注册git使用的邮箱"
